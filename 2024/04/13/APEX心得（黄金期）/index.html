# APEX 心得（黄金期）

# 思路

关于思路部分，我认为这是我三部分中目前最好的部分。得益于平时观摩复盘视频，对点位，身位控制还有进攻思路都有一定的理解。但是目前存在两个问题，一个问题是我的枪法支撑不住我想实现的思路，即使是站在高打低的头皮位，我也很难跟住枪。第二个问题是对于自己的复盘看得很少，基本都是在剪下饭视频，但也是受限于设备，之后看看 n 卡回放能不能拉长一点吧。

# 枪法

最菜的一部分，定位还可以，跟枪完全跟不住。目前分析原因是灵敏度太高了（1600*4.79）试着慢慢降下去吧。还有就是练枪，特别感谢玉玉哥教我练枪。目前网格射击稳定在 6W+，最高打到 7W1。

# 身法

会一点蹬墙跳和 ts，但是把这两种结合到一起就跟残疾一样，总是搓不出来。绳索方面学了学 mantal jump，但是成功率不是很高，现在也就是五五开的成功率吧。特别感谢狐梦大佬教我身法，那几个小时让我受益匪浅，最大的收获就是会转角蹬墙跳 + ts 转向过弯了。
